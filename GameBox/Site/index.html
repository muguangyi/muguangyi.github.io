GameBox是Unity开发中间件。GameBox将各种模块定义成一个个服务，独立编译成DLL，采用插件框架加载，通过抽象化的接口提供功能。

version 1.0

Unity v5.3+ 快速接入

将下载的ServiceCenter.Editor.dll放在Unity项目中的Editor目录
在Unity的GameBox菜单下选择ServiceCenter，在弹出的对话框中点击Create
在更新的服务列表中选择需要的模块，然后点击Update
在Hierarchy下创建一个GameObject，展开GameBox/ServiceCenter模块，将ServicePlayer挂到该GameObject上
在Project下展开需要的模块文件，将对应的Installer挂在之前的GameObject上

快速入门

当你所需要的各种服务已经在Unity中设置完毕，只需在代码中获取需要的服务接口，然后就可以直接使用接口提供的各种功能了。

  • 确保所依赖的服务已经准备完毕
  • 获取对应的服务接口
异步获取服务接口
// 获取资产服务接口
new ServiceTask<IAssetManager>().Start().Continue(task =>
{
    var assetManager = task.Result as IAssetManager;
    return null;
});
