Gaint GameBox Unity中间件

简介

巨人GameBox Unity中间件是为Unity开发的基于插件思想的客户端中间件框架，所有的模块都是以DLL形式提供，功能都是以interface对外暴露，在语言层面规范访问的边界。所有的模块都是以Service定义，通过ServiceCenter获取到相应的Service，然后转换成对应的Service接口，就可以直接使用Service对外暴露的各种方法。

快速入门

初始化

  • 在Unity项目的Assets目录下创建Editor目录
  • 将GameBox/ServiceCenter.Editor.dll下载放在Editor目录
  • 切换回Unity，在GameBox菜单中选择ServiceCenter.Editor
  • 在弹出的工具窗口点击“Create”按钮
  • 在更新后的界面勾选需要的组件，并点击“Update”按钮

配置

  • 在场景中创建一个GameObject对象
  • 将ServiceCenter下的ServicePlayer组件挂接到该GameObject上
  • 将需要的其余组件依次挂接到该GameObject上即可

使用

异步获取服务

// 通过服务接口获取资产管理服务
new ServiceTask<IAssetManager>().Start().Continue(task =>
{
var assetManager = task.Result as IAssetManager;
return null;
});
// 通过服务ID获取资产管理服务
new ServiceTask("com.giant.serivce.assetmanager").Start().Continue(task =>
{
var assetManager = task.Result as IAssetManager;
return null;
});

同步获取服务

获取前需确保相关服务已经装载完毕。

// 使用ServiceCenter通过服务接口获取资产管理服务
var assetManager = ServiceCenter.GetService<IAssetManager>();
// 使用ServiceCenter通过服务ID获取资产管理服务
var assetManager = ServiceCenter.GetService("com.giant.service.assetmanager") as IAssetManager;
